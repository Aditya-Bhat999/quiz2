1. Which of the following is NOT a type of AI?

A) Narrow AI
B) General AI
C) Super AI
D) Manual AI

Answer: D) Manual AI

2. Which AI approach tries to mimic the way the human brain works?

A) Expert Systems
B) Neural Networks
C) Genetic Algorithms
D) Fuzzy Logic

Answer: B) Neural Networks

3. What does NLP in AI stand for?

A) Natural Learning Process
B) Neural Language Processing
C) Natural Language Processing
D) Non-Linear Programming

Answer: C) Natural Language Processing

4. Which programming language is most commonly used for AI today?

A) C
B) Python
C) JavaScript
D) PHP

Answer: B) Python

5. In Machine Learning, what is overfitting?

A) Model performs well on both training and test data
B) Model performs poorly on training data
C) Model performs well on training data but poorly on test data
D) Model ignores training data

Answer: C) Model performs well on training data but poorly on test data

6. Which of the following is an application of AI?

A) Self-driving cars
B) Virtual assistants (Siri, Alexa)
C) Fraud detection in banking
D) All of the above

Answer: D) All of the above

7. Who is considered the “father of Artificial Intelligence”?

A) Charles Babbage
B) Alan Turing
C) John McCarthy
D) Marvin Minsky

Answer: C) John McCarthy

8. Which of these learning types allows machines to learn from labeled data?

A) Reinforcement Learning
B) Supervised Learning
C) Unsupervised Learning
D) Deep Learning

Answer: B) Supervised Learning

9. What is the main function of a chatbot?

A) Play music
B) Automate conversations with humans
C) Recognize objects in images
D) Analyze big data

Answer: B) Automate conversations with humans

10. In Deep Learning, what does CNN stand for?

A) Central Neural Network
B) Convolutional Neural Network
C) Conditional Neural Net
D) Connected Neural Network

Answer: B) Convolutional Neural Network
